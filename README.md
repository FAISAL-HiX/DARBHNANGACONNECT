## 🚀 **Quick Start**

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/darbhangaconnect.git
   cd darbhangaconnect
   ```

2. **Start local server:**
   ```bash
   # Using Python
   python3 -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

3. **Open in browser:**
   ```
   http://localhost:8000
   ```

## 💳 **Razorpay Setup**

1. Create account at [razorpay.com](https://razorpay.com)
2. Get your API keys from dashboard
3. Replace `YOUR_RAZORPAY_KEY_HERE` in `payment.html`
4. Test with provided test cards
